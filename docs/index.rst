===========================================================
Sermin - Python framework for stateful system configuration
===========================================================

Sermin is a Python framework for defining and applying system states.

Read the quickstart in the :doc:`introduction <introduction>` to get a feel for
how it works, dive straight into the :doc:`installation <installation>`,
:doc:`usage <usage>` and :doc:`blueprints <blueprints>` documentation to see
the details, or look up state arguments in the :ref:`API reference <modindex>`.
